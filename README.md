# 📌 Printrest (Pinterest Clone)

A simple Pinterest-style web application built with **Node.js, Express, MongoDB, EJS, and Bootstrap**.  
Users can create posts, like, save, and view other users’ profiles.

---

## 🚀 Features

- 🔐 User Authentication (Login / Register)
- 🖼️ Create and upload posts
- ❤️ Like posts
- ⭐ Save posts
- 👤 Public user profiles
- 🔍 Search posts
- 📱 Responsive UI (Bootstrap)
- 🧑 Dynamic navbar (login/logout state)

---

Operations 

This is the printrest clone app that can perform the follwing tasks.
1) Login and Sign up users.
2) User can post images with title and  descpription.
3) User can like and saves posts.
4) User can create and delete posts.
5) User can search posts by name.
6) User can access ohers profie.

## 🛠️ Tech Stack

- Node.js
- Express.js
- MongoDB + Mongoose
- EJS (Templating Engine)
- Passport.js (Authentication)
- Bootstrap 5
